## Claude Desktop

Connect SecureLend to Claude Desktop in under 2 minutes.

### Prerequisites

- [Claude Desktop](https://claude.ai/download) installed (available for Mac, Windows, Linux)
- Text editor to edit configuration file
- Internet connection

### Step 1: Locate Configuration File

Find your Claude Desktop configuration file based on your operating system:

<Tabs>
  <Tab title="macOS">
    ```bash
    ~/Library/Application Support/Claude/claude_desktop_config.json
    ```

    **Quick Access:**
    ```bash
    # Open in default editor
    open ~/Library/Application\ Support/Claude/claude_desktop_config.json

    # Or navigate in Finder
    open ~/Library/Application\ Support/Claude/
    ```

  </Tab>
  
  <Tab title="Windows">
    ```bash
    %APPDATA%\Claude\claude_desktop_config.json
    ```

    **Quick Access:**
    1. Press `Win + R`
    2. Type: `%APPDATA%\Claude`
    3. Press Enter
    4. Open `claude_desktop_config.json` in Notepad

  </Tab>
  
  <Tab title="Linux">
    ```bash
    ~/.config/Claude/claude_desktop_config.json
    ```

    **Quick Access:**
    ```bash
    # Open in default editor
    xdg-open ~/.config/Claude/claude_desktop_config.json

    # Or with nano
    nano ~/.config/Claude/claude_desktop_config.json
    ```

  </Tab>
</Tabs>

### Step 2: Add SecureLend Server

Open the configuration file and add SecureLend to the `mcpServers` section:

<CodeGroup>
```json New Configuration
{
  "mcpServers": {
    "securelend": {
      "url": "https://mcp.securelend.ai/mcp"
    }
  }
}
```

```json Existing MCP Servers
{
  "mcpServers": {
    "securelend": {
      "url": "https://mcp.securelend.ai/mcp"
    },
    "other-server": {
      "url": "https://example.com/mcp"
    }
  }
}
```

</CodeGroup>

<Warning>
  Ensure your JSON syntax is valid. Missing commas or brackets will prevent
  Claude Desktop from starting.
</Warning>

### Step 3: Restart Claude Desktop

1. **Quit Claude Desktop completely**
   - macOS: `Cmd + Q` or Quit from menu bar
   - Windows: Right-click system tray icon â†’ Exit
   - Linux: Close all windows and background processes

2. **Reopen Claude Desktop**

3. **Wait for MCP servers to connect** (usually 2-3 seconds)

### Step 4: Verify Connection

Open a new conversation in Claude Desktop and try:

<CodeGroup>
```text Business Loans
Compare business loans for $200,000 for equipment. 
My business has $1.2M annual revenue and a 720 credit score.
```

```text Personal Loans
Compare personal loans for $25,000 for debt consolidation
with a credit score of 720.
```

```text Mortgage Calculator
What would my monthly payment be on a $400,000 home with
20% down at 6.5% interest over 30 years?
```

</CodeGroup>

If working correctly, Claude will use the SecureLend tools and return loan offers or calculations.

<Check>
  **Success!** You should see Claude using the `compare_business_loans`,
  `compare_personal_loans`, or `calculate_mortgage_payment` tools.
</Check>

## TypeScript SDK

For programmatic access, use our TypeScript SDK:

```bash
npm install @securelend/mcp-client
# or
pnpm add @securelend/mcp-client
```

```typescript
import { SecureLendMCP } from "@securelend/mcp-client";

const client = new SecureLendMCP();

// Compare business loans
const loans = await client.compareBusinessLoans({
  loanAmount: 200000,
  purpose: "equipment",
  annualRevenue: 1200000,
  creditScore: 720,
});

console.log(`Found ${loans.offers.length} offers`);
loans.offers.forEach((offer) => {
  console.log(`${offer.lenderName}: ${offer.interestRate}% APR`);
});
```

**Full SDK documentation:** [/sdk/javascript](/sdk/javascript)

**SDK Repository:** [github.com/SecureLend/sdk](https://github.com/SecureLend/sdk)

## Troubleshooting

### Claude Desktop Issues

<AccordionGroup>
  <Accordion title="Tools not appearing in Claude">
    **Possible causes:**
    - Configuration file syntax error (invalid JSON)
    - Claude Desktop not fully restarted
    - MCP server URL incorrect

    **Solutions:**
    1. Validate JSON syntax with [jsonlint.com](https://jsonlint.com)
    2. Ensure URL is exactly: `https://mcp.securelend.ai/mcp`
    3. Quit Claude Desktop completely (check system tray/menu bar)
    4. Reopen and wait 5 seconds for MCP connection

  </Accordion>

  <Accordion title="Connection timeout errors">
    **Possible causes:**
    - No internet connection
    - Firewall blocking HTTPS traffic
    - Corporate proxy interfering

    **Solutions:**
    1. Test connection: Open `https://mcp.securelend.ai/mcp` in browser
    2. Check firewall settings allow HTTPS to mcp.securelend.ai
    3. If behind corporate proxy, contact IT about MCP access
    4. Check [status.securelend.ai](https://status.securelend.ai) for service status

  </Accordion>

  <Accordion title="Tools return errors">
    **Possible causes:**
    - Invalid parameters passed to tools
    - MCP server experiencing issues
    - Rate limit exceeded

    **Solutions:**
    1. Check parameter requirements in [Tool Reference](/mcp/tools)
    2. Verify [status.securelend.ai](https://status.securelend.ai)
    3. Wait 60 seconds if rate limited
    4. Contact support@securelend.ai if issues persist

  </Accordion>

  <Accordion title="Configuration file not found">
    **Possible causes:**
    - Claude Desktop not installed
    - Non-standard installation location
    - Configuration file not yet created

    **Solutions:**
    1. Verify Claude Desktop is installed
    2. Create the configuration file manually if it doesn't exist:

    ```bash
    # macOS
    mkdir -p ~/Library/Application\ Support/Claude
    echo '{"mcpServers":{}}' > ~/Library/Application\ Support/Claude/claude_desktop_config.json

    # Windows (PowerShell)
    New-Item -ItemType File -Path "$env:APPDATA\Claude\claude_desktop_config.json" -Force

    # Linux
    mkdir -p ~/.config/Claude
    echo '{"mcpServers":{}}' > ~/.config/Claude/claude_desktop_config.json
    ```

    3. Then add SecureLend server configuration

  </Accordion>
</AccordionGroup>

### ChatGPT Issues

<AccordionGroup>
  <Accordion title="Can't find SecureLend GPT">
    **Solutions:**
    1. Ensure you have ChatGPT Plus or Enterprise
    2. Search for exact name: "SecureLend Financial Services"
    3. Check if GPT is available in your region
    4. Contact support@securelend.ai if unavailable
  </Accordion>

  <Accordion title="GPT returns generic responses">
    **Possible causes:**
    - Not actually using SecureLend GPT
    - MCP server connection failed

    **Solutions:**
    1. Verify you're in the SecureLend GPT conversation
    2. Look for "SecureLend Financial Services" at top of chat
    3. Try starting a new conversation with the GPT

  </Accordion>
</AccordionGroup>

### General Issues

<AccordionGroup>
  <Accordion title="Results seem outdated or incorrect">
    **Solutions:**
    1. Loan offers update in real-time - refresh query
    2. Check [status.securelend.ai](https://status.securelend.ai) for service status
    3. Verify parameters match lender requirements
    4. Contact support@securelend.ai with specific examples
  </Accordion>

  <Accordion title="Need help with specific lender">
    **Solutions:**
    1. After getting comparison results, use `get_offer` to submit application
    2. Use `track_offer_status` to check application progress
    3. Contact specific lender directly for detailed questions
    4. Email support@securelend.ai for integration help
  </Accordion>
</AccordionGroup>
